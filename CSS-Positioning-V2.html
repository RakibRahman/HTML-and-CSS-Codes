Relative Position

The relative value for the position property allows elements to appear within the normal flow a page, leaving space for an element as intended while not allowing other elements to flow around it.

<div>
<div class=
"offset">
left: 20px;
top: 20px;
<div>

Absolute Position

The absolute value for the position property is different from the relative value in that an element with a position value of absolute will not appear within the normal flow of a document, and the original space and position of the absolutely positioned element will not be preserved.

<section>
position: relative;
<div class="offset">
position: absolute;
right: 20px;
top: 20px;

Absolutely positioned elements are moved in relation to their closest relatively positioned parent element.
